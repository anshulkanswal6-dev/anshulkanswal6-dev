### 🛡️ AEGIS
**AI-Powered Vibe-Coding Platform for Autonomous On-Chain Agents**

**Problem Solved:** Enables non-technical users to create and deploy autonomous AI agents capable of executing blockchain transactions using natural language, removing the smart-contract-literacy barrier to on-chain automation.

**Architecture:** Agent orchestration layer + wallet integration layer + task execution engine, built for the Monad ecosystem (MON Testnet).

**Tech Stack:** `Python` `web3.py` `Flask` `Solidity` `Monad Testnet`

**Key Features:**
- Natural-language-to-agent-action pipeline
- Wallet integration for autonomous transaction signing
- Modular task execution workflows

**Engineering Considerations:**
- 🔐 Secure wallet handling for agent-initiated transactions
- 📈 Designed for extensibility across additional EVM chains
- ⚙️ Health, claim, and balance endpoints for reliable agent operations

### 💠 Twin
**MCP-Powered Agentic Finance Layer**

**Problem Solved:** Provides secure execution infrastructure between AI agents (Claude, Cursor, and other MCP-compatible agents) and on-chain financial operations — while preserving full user custody.

**Architecture:** Agent Wallets + AI Sessions + ticket-based orchestration engine sitting between MCP agents and Ethereum-based financial rails.

**Tech Stack:** `MCP Protocol` `Ethers.js` `Ethereum` `Node.js`

**Key Features:**
- Autonomous payments and scheduled transactions via natural language
- Wallet analysis and financial workflow management
- Ticket-based orchestration for auditable agent actions

**Engineering Considerations:**
- 🔐 Custody-preserving execution model — agents act, users retain control
- 🧩 Built for interoperability across MCP-compatible AI clients
- ⚖️ Designed around security-first financial automation
